# java-collections-framework

# Welcome!
This repository is my personal journey of mastering the Java Collections Framework (JCF) — from the basics to advanced concepts — committed step-by-step for easy learning and reference.

# What’s Inside?

Core Interfaces – Collection, List, Set, Queue, Map
Implementations – ArrayList, LinkedList, HashSet, TreeSet, HashMap, TreeMap, etc.
Utility Classes – Collections, Arrays
Advanced Topics – Iterators, Comparators, Streams with Collections
Real-World Examples – Practical use cases for each collection type
